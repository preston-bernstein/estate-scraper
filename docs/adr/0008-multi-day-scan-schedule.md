# 0008 — Scan schedule: single weekly run, Thursday at 02:00 local

The original Friday 1 am schedule missed Thursday morning (the first estate sale day) and ran too close to Saturday to be useful. One scan on Thursday at 2:00 AM local time produces fresh data by the time the user logs in Thursday morning; the same dataset serves Friday and Saturday browsing since estate sale listings stabilize by Wednesday night. 2:00 AM is chosen over 11:00 PM because the user is at the computer until around midnight; 2 AM ensures the machine is idle. `caffeinate -s` in run-scan.sh prevents macOS from sleeping and silently skipping the LaunchAgent. LaunchAgent Weekday 4 (Thursday), Hour 2. If the machine is fully powered off the scan is missed — accepted for a homelab tool. The scan writes to shared `sales` and `findings` tables with no per-user context; all users see the same underlying data and personalize via Hunts at browse time.
