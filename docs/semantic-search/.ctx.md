## CONTEXT.md
Domain glossary (opinionated, enforced vocabulary). Key terms:
- Sale — one estatesales.net listing (date range, address, images).
- Finding — an Image the vision model flagged, plus its plain-text description.
- Image — every analyzed photo; carries the durable frozen SigLIP embedding + thumbnail.
- Item — one identified object within a Finding (finding_items row: maker, category, era, desirability).
- Hunt — a person's saved keyword filter over Findings.
- Scan — weekly run; results accumulate, never overwrite.
- Standout — a Finding with scoreFinding() >= 4, shown in Discover featured scroll.
- Desirability — high/med/low, distinct from identification and price.
- Identifier scope — tells the user which Sales are worth attending; NOT a deal-finder (no prices).

## ADRs (titles only)
- 0016 — generator provenance, frozen embedding model (embed_model/embed_dim stamped; change = re-embed migration).
- 0017 — SigLIP image embeddings (siglip-so400m-patch14-384, 1152-dim) over OpenAI /embeddings wire, computed post-scan from thumbnail; EMBED_DIM reject-wrong-length guard; cosine query is the Tier-1 job.
- 0018 — deterministic finding_items extraction; category is a CLOSED vocab of 15 (seating, tables, case_goods, beds, lighting, clocks, art, ceramics_glass, silver, jewelry_watches, electronics, games_toys, instruments, kitsch, other); lexicon (lib/lexicon.ts) is sole maker authority.
